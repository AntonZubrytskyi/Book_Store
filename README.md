
# Book Store

Technologies:
- ReactJS
- TypeScript
- Redux Toolkit
- React Router
- React Table
- React OAuth Google
- React Form
- Axios
- React Hooks
- CSS-Modules / SCSS

## Running the JSON server
1. Open a new terminal tab
2. ```npm run server```


## Running the application
1. Open a new terminal tab
2. ```npm start```


## Running the tests
1. Open a new terminal tab
2. ```npm test```

App URL:
```http://localhost:3000```
Server URL:
```http://localhost:5000```
